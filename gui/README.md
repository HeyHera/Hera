## User Interface
Using [Kivy](https://kivy.org/)
And using [KivyMD](https://kivymd.readthedocs.io/en/latest/)


## External Dependencies
### Installing Kivy
```
pip install "kivy[full]" kivy_examples
```
### Installing KivyMD
```
pip3 install kivymd==0.104.2
```
### Installing mpg321
```
sudo apt install mpg321
```
